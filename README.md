# 분골사 팀편성

스크린골프 모임을 위한 간단한 방·팀 추첨 웹앱입니다.

- 참가자 이름과 핸디캡을 입력하면 팀을 자동으로 편성합니다.
- 랜덤 배정 / 실력 균등(핸디캡 기준) 두 가지 모드를 지원합니다.
- Vite + React + TypeScript 기반.

## 시작하기

```bash
npm install
npm run dev
```

브라우저에서 http://localhost:5173 에 접속하세요.

## 빌드

```bash
npm run build
```

생성된 `dist` 폴더를 Vercel 등 원하는 곳에 배포할 수 있습니다.
